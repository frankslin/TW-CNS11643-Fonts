## CNS11643中文標準交換碼全字庫(簡稱全字庫)

zip壓縮檔，內容包含全字庫字型、屬性資料及中文碼對照表三部分，其中全字庫字型提供正宋體及正楷體2種；屬性資料則涵蓋注音、倉頡、筆畫、部首、拼音、部件及筆順等7種；中文碼對照表則提供BIG5、Unicode、電信碼、地政自造字、財稅內碼、稅務碼及工商自造字等7種中文內碼對照。
